== Description ==

CiviCRM Directory creates a publicly-viewable directory from data submitted to CiviCRM.

### Requirements

This plugin requires a minimum of *CiviCRM 4.7*

### Plugin Development

This plugin is in active development. For feature requests and bug reports (or if you're a plugin author and want to contribute) please visit the plugin's [GitHub repository](https://github.com/christianwach/civicrm-directory).



== Installation ==

1. Extract the plugin archive
1. Upload plugin files to your `/wp-content/plugins/` directory
1. Make sure CiviCRM is activated and properly configured
1. Activate the plugin through the 'Plugins' menu in WordPress
